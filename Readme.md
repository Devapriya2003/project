# Multi-Disease Prediction System

A web-based machine learning application that predicts multiple diseases using clinical parameters.

## Diseases Included

- Heart Disease
- Lung Disease (COPD)
- Kidney Disease
- Liver Disease

## Technologies Used

- Python
- Flask
- Machine Learning (Scikit-learn)
- HTML, CSS, Bootstrap
- SQLite Database

## System Architecture

The system follows this workflow:

User Input → Data Preprocessing → Machine Learning Model → Disease Prediction → Result Display → Database Storage

## Features

- User Login and Registration
- Disease Prediction Dashboard
- Prediction History Storage
- Multiple ML Models Integrated

## Datasets Used

- UCI Heart Disease Dataset
- COPD Dataset
- Chronic Kidney Disease Dataset
- Indian Liver Patient Dataset (ILPD)

## How to Run the Project

1. Clone the repository
git clone https://github.com/Devapriya2003/project.git

2. Navigate to the project folder
cd project


3. Install required libraries
pip install -r requirements.txt


4. Run the Flask application
python app.py


5. Open browser and go to
http://127.0.0.1:5000
